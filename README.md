# fontinstalled

Detect using JavaScript whether a font-face is installed locally. Inspired heavily by Remy Sharp's [blog post](http://remysharp.com/2008/07/08/how-to-detect-if-a-font-is-installed-only-using-javascript/).


## Demo

1. Install the [Seymour One](http://www.fontsc.com/font/download/seymour-one) font on your computer.
2. Load the [demo page](http://cvan.github.io/fontinstalled/demo/).
